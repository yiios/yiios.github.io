修改密码和 API Key 手把手教程

· 2023-05-10 ·

下单地址：https://s.yiios.com

整个流程均需要排🪜，全局代理，建议使用 美国、新加坡、日本地址。如果某个页面不允许访问，就换地区。

修改邮箱密码

登录邮箱面板：https://mailline.org/admin/login

点击进入密码修改页面：

输入新密码后，点保存。密码修改完成。

修改 ChatGPT 密码

登录ChatGPT:https://chat.openai.com/chat


选择忘记密码接下来按指引操作即可。

登录邮箱：https://mailline.org/，可以查看密码修改邮件。

修改APIKEY

访问地址：https://platform.openai.com/
这里需要登录，使用修改后的密码进行登录。

找不到入口的直接访问地址：https://platform.openai.com/account/api-keys


先生成一个新的key，再将旧的key删除即可
